# Known-flaky/known-broken example scripts, one relative path per line.
#
# The sweep (scripts/sweep_examples.sh) treats a FAIL or TIMEOUT on any path
# listed here as expected, not a CI-breaking regression. Every entry has a
# reason so this stays honest about what's actually broken vs. what's simply
# infeasible to run unattended in a sweep (this is a "no silent caps"
# allowlist, not a way to sweep failures under the rug — if a reason below
# stops being true, delete the line so the sweep enforces it again).
#
# Starts an HTTP/gRPC server and blocks forever waiting for a client; there
# is no client in this sweep to make it exit.
examples/example6-web.r2

# Makes real outbound network calls to external hosts; fails or times out
# depending on network conditions/reachability at sweep time, not a code bug.
examples/example9-client-http.r2
examples/simple-p3p4-demo.r2

# Uses examples/testing/*.r2's describe/it/assert DSL, which is only
# provided when run through cmd/r2test (see pkg/r2test), not the plain
# interpreter this sweep uses. Expected to fail here by design.
examples/testing/basic_test.r2
examples/testing/advanced_test.r2
